# Matchday Manager V4 - Stage 1

This is the first real cloud-sync stage.

## Working now

- Supabase login
- Cloud-synced players
- Cloud-synced app title/logo/theme
- Mobile-friendly layout

## Setup

1. Run `supabase_setup_stage1.sql` in Supabase SQL Editor.
2. Edit `config.js`.
3. Upload these files to your GitHub repo.
4. Vercel will deploy automatically.

- `index.html`
