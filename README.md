# Matchday Manager V4 Cloud Sync

1. Run supabase_setup.sql in Supabase SQL Editor.
2. In Supabase Settings > API Keys, copy your Project URL and Publishable key.
3. Paste them into config.js.
4. Upload this whole folder to Vercel.
5. Open the URL, create an account, and log in.

Do not use the secret key in config.js.
